## [1.3.0] 2021-02-25

### Added

## [1.2.0] 2020-10-28

### Added

* Option to pick the C++ ISO standard to use for the compiler.
* Option to process or skip flags.
* Keyboard shortcut `shift + cmd + R` to **build and run** code in the active viewer.
